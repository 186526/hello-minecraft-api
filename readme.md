## Installation

You need to install node.js >= 12 && yarn first.

```bash
git clone https://github.com/186526/hello-minecraft-api.git
## or git clone https://hub.fastgit.org/186526/hello-minecraft-api

cd hello-minecraft-api

yarn install

yarn start
```
